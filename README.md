# olist-ecommerce-analysis
Comprehensive e-commerce data analysis using SQL. Analyzing sales trends, logistics performance, and customer satisfaction correlations on the Olist dataset.

## Dashboard Visualization
![Olist Dashboard](Olist%20Ecommerce%20Dashboard.png)

### Key Insights from the Analysis
* **Delivery Performance:** The analysis clearly shows that as the average delivery time exceeds **19.5 days**, customer satisfaction drops significantly to 1 star.
* **Payment Methods:** Credit card is the most preferred payment method, indicating a high demand for installment options.
* **Logistics:** Certain regions in Brazil show higher logistics risks with increased lead times (highlighted in the map).
